The VAT codes list stores value added taxes that your business uses in purchase and sales transactions.

The VAT codes list is only available if the VAT check box is selected in Accounting settings. 

To add a VAT code

  1. In the Accounting section, click VAT codes, and then click Create.
  2. Enter or select the following:
    • Code. The VAT code. It helps you quickly select the VAT in various documents.
    • Description. The VAT description.
  3. In both Sales and Purchase panes, enter or select the following:
    • Incl. rate. The VAT rate if the price includes VAT (for example, 20%).
    • Excl. rate. The VAT rate if the price does not include VAT (for example, 25%).
    • Account. The VAT account used for automatic transaction posting.
  4. Click Write and close.
