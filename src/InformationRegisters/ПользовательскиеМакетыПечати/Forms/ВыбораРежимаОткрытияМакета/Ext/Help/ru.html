Для открытия макета используйте:

  • клавишу Enter 
  • двойной щелчок левой кнопкой мыши
  • команду "Открыть"

 

Варианты открытия макета:

Только для просмотра Макет не будет помечен как занятый Вами для редактирования и откроется внешним приложением в режиме "только просмотр"
Для редактирования Макет будет помечен как занятый Вами для редактирования и откроется внешним приложением с возможностью внесения и сохранения изменений
